# CineWild

# Installation
```
npm install -g yarn
cd back
npm install
cd ../front
yarn install
```

# Lancement backend
```
cd back
npm start
```

# Lancement frontend 
```
cd front
yarn start
```

# Infos sur les ports
- Le backend écoute sur le port 3000
- Le frontend écoute sur le port 3001
- Pour appeller le backend depuis le frontend : http://localhost:3001/api et non pas http://localhost:3001 (il faut que les 2 soient lancés)
